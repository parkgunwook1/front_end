문자열 속성 선택자

        속성 값의 문자열을 확인하여 스타일 적용하는 방식의 선택자

        1) 선택자(속성명~=속성값) {css코드}
        -> 띄어쓰기로 구분되어 있는 여러 속성 값이 작성된 속성 중
           속성값이 특정값을 단어로 포함하는 요소를 선택

        2) 선택자[속성명 |= 속성값] {css코드}
        -> 속성값이 특정값을 단어로 포함하는 요소를 선택
           단, "-" 기호로 구분, "-"앞의 내용이 동일해야한다. 

        3) 선택자[속성명 ^= "속성값"]{CSS 코드} (^ : 캐럿)
        -> 속성값이 속성값으로 시작하는 요소를 선택   

        4) 선택자[속성명 $= "속성값"] {CSS코드}
        -> 속성값이 특정값으로 끝나는 요소를 선택

        5) 선택자[속성명 *= 속성값1] {CSS코드}
        -> 속성값이 특정 값을 포함하는 요소선택
