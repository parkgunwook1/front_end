CSS(Cascading Style Sheets)

        마크업 언어가 실제로 화면에
        표시되는 방법을 기술하는 언어.
        (HTML 코드가 화면에 어떤 스타일로 보여질지 지정하는 언어)

        W3C에서 CSS를 웹 표준으로 지정

CSS Selector(선택자)

        Selector란?
        현재 HTML문서 내에서 특정한 요소를 선택하는 문자 또는 문자열

        스타일을 적용하고자 하는 요소를 선택하는 방법

태그 선택자

        현재 HTML 문서 내에 같은 태그들을 선택하는 선택자

        [작성법]

        태그명 {css 코드;}
strong 태그에 작성한 내용
span 태그에 작성된 내용입니다.
  • strong 태그에 작성한 내용
  • span 태그에 작성된 내용입니다.

id 선택자 (#id속성값)

        HTML 문서 내에서 id속성의 속성값과 값이 일치하는 요소를 선택하는 선택자다.

        (중요!) id 속성값은 페이지 내에서 유일해야한다. 
                -> 중복되는 id 속성 값이 있으면 안된다.

        [작성법]
        #id속성값(css코드;)
  1. id 선택자 테스트1
  2. id 선택자 테스트2
  3. id 선택자 테스트3
  4. id 선택자 테스트4
  5. id 선택자 테스트5

class 선택자(.calss속성값)

        HTML 문서 내에서 같은 class 속성 값을 가지는 태그를 모두 선택하는 선택자

        [작성법]
        .class속성값 {css코드}
