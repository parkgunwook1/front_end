글자 관련 스타일

color : 글자색을 지정하는 속성

        선태자 {
            color : 색상명(영문) | 16진수 숫자 rgb(#fff , #ffff)
                | rgb(255,255,255) | rgba(255,255,255,1)
                | hsl(360,100,100) | hsla(360,100,100,1)

                * rgba에서 a는 투명도(0 : 투명, 1:불투명)
                * hs1(색조, 채도, 밝기)
                * 6개의 색상 지정 방법은 색과 관련된 모든 css속성에서 사용 가능
        }
